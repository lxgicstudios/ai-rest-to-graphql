Convert REST API routes to a GraphQL schema

## Install

```bash
npm install -g ai-rest-to-graphql
```

## Usage

```bash
npx ai-rest-to-graphql ./path/to/file
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
